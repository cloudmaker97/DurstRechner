    Export / Import
    Exportieren und Importieren

    Die Daten können im JSON-Format exportiert und importiert werden. Um Speicherplatz zu sparen, sollte das Bild im Format 1:1 vorliegen. Ideal ist eine Bildgröße von 250x250 Pixel. Wenn ein Bild über eine URL bereitgestellt wird, lädt die Software das Bild herunter und speichert es in der JSON-Datei als Base64-kodierte Version. Die ursprüngliche URL wird dabei nicht mehr gespeichert.
